Erweiterte Suche
So funktioniert die erweiterte Suche
Mit der Angabe von Operatoren können Sie die Suche besser kontrollieren und erhalten dadurch bessere Antworten. Die folgenden Suchoptionen können jeweils direkt in das Suchfeld eingegeben werden.
Die Grundlagen
In das Suchfeld können Sie Zahlen und/oder Buchstaben sowie Ausdrücke eingeben, z.B. 'Müller', '> 500' or '>= Meier' und dann
• auf den Text der Tabellenspalte klicken, um Objekte zu finden, die für das entsprechende Attribut passende Werte aufweisen (bei dieser Suche werden alle Objekte durchsucht, unabhängig von evtl. vorherigen Suchen oder gesetzten Filtern; bestehende Suchresultate gehen verloren Neue Suche
• auf den [+] Knopf klicken, um Objekte zu finden, die für das entsprechende Attribut passende Werte aufweisen (nur diejenigen Objekte werden durchsucht, welche sich im aktuellen Filter oder im vorherigen Suchresultat befinden, d.h. diese Suche 'hinzufügen' bewirkt, dass das neue Resultatset eingeschränkter sein wird) Suche hinzufügen
Operatoren
Die folgenden Operatoren werden unterstützt:
  • tönt ähnlich wie (*), voreingestellte Suche (Default) für Zeichenketten
  • ist ähnlich wie (%)
  • gleich (=), voreingestellte Suche (Default) für boolesche Werte, Zahlen und Codes
  • nicht gleich (<>, !=)
  • kleiner als (<)
  • kleiner als oder gleich wie (<=)
  • grösser als (>)
  • grösser als oder gleich wie (>=)
"UND" Suchen
Um Objekte zu finden, die gleichzeitig mehr als eine Bedingung erfüllen, können Bedingungen mit dem Zeichen "&" verknüpft werden. Um zum Beispiel Objekte mit Werten zwischen 500 und 1000 zu finden (d.h. grösser als 500 und kleiner als 1000), wird in das Suchfeld folgendes eingegeben: '>500 & <1000'
"ODER" Searches
Um Objekte zu finden, die entweder das Wort A oder das Wort B enthalten, werden die beiden Worte mit einem Strichpunkt ";" getrennt. Um zum Beispiel Kontakte mit dem Namen 'Meier' ODER 'Müller' zu finden, wird in das Suchfeld folgendes eingegeben: '= Meier; Müller'
Suchen kombinieren
Suchen können fast ohne Einschränkungen kombiniert werde. Um zum Beispiel Objekte mit einem Wert kleiner als 1000, aber verschieden von 500 und verschieden von 600 zu finden, geben Sie folgendes ein in das Suchfeld:'<>500; 600 & <1000'
Kombinierte Suchen werden nach den folgenden Regeln angewendet:
  • searchExpr ::= filterValue { & filterValue }
    (--> die Zeichenkette wird jeweils bei einem & aufgeteilt)
  • filterValue ::= [operator] value { ; value }
    (--> die Zeichenkette wird jeweils bei einem ; aufgeteilt)
