ActiveX

Utilizarea funcţionalităţilor bazate pe controalele ActiveX necesită ca

  • navigatorul de web să fie capabil de a gestiona controale ActiveX (ex. Internet Explorer în sistem MS Windows) şi
  • configurările de securitate să permită încărcarea/execuţia controalelor ActiveX

Dependent de funcţionalitatea utilizată, suplimentar ar putea fi solicitate anumite pachete software :

Funcţionalitate Necesar Sugestii adiţionale
MS Word XML Merge instalarea MS Word 2000 sau mai nouă, în configuraţie client Trusted Sites, ActiveX Permissions
MS Excel XML Merge instalarea MS Excel 2000 sau mai nouă, în configuraţie client  
XML Export to MS Outlook instalarea MS Outlook 2000 sau mai nouă, în configuraţie client  
XML Import from MS Outlook instalarea MS Outlook 2000 sau mai nouă, în configuraţie client  

Site-uri Sigure (Internet Explorer)

Din motive de securitate nu e recomandată activarea controalelor ActiveX pentru Zona Internet. E mai bine să adăugaţi site-ul care rulează openCRX în lista site-urilor sigure Zona Site-urilor de Încredere sau  Zona Intranet-ului Local (doar dacă site-ul este în reţeaua locală) după care  activaţi controalele ActiveX pentru zona respectivă. Mai jos sunt exemplificate modalitaţile de adăugare a openCRX demo server (demo.opencrx.org) la zona site-urilor de încredere şi modul de adăugare a unui server local openCRX (10.1.1.20) la zona reţelei locale.
