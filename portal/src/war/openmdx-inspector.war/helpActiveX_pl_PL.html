ActiveX

Uzywanie aplikacji na podstawie ActiveX controls wymaga ze

  • twoje przegladarka moze przetwarzac ActiveX controls (e.g. Internet Explorer on MS Windows) i
  • ze ustawienie zabezpieczenia pozwala ladowac/wykonywac ActiveX controls

Zalezy od aplikacji, niektore pakiety oprogramowania trzeciej strony sa wymagane:

Aplikacja Wymagania Wskazowka
MS Word XML Merge client-based installation of MS Word 2000 alb nowsze Trusted Sites, ActiveX Permissions
MS Excel XML Merge client-based installation of MS Excel 2000 albo nowsze  
XML Export to MS Outlook client-based installation of MS Outlook 2000 albo nowsze  
XML Import from MS Outlook client-based installation of MS Outlook 2000 albo nowsze  

Strony godne zaufania (Internet Explorer)

Z powodu bezpieczenstwa nie zalecamy zeby aktywowanie ActiveX controls dla strefy internetowej. Jest najlepiej dodac strone na ktorej jest openCRX do strefy zabezpieczonej albo do lokalnej strefy internetowej (tylko jezeli strona znajduje sie na Intranet) i dopiero wtedy wlacz ActiveX controls dla tej strefy. Ponizej sa podane przyklady jak dolaczyc openCRX demo server(demo.opencrx.org) do strefy zobezpieczonej stron i jak mozna dolaczyc openCRX server (10.1.1.20) do lakalnego intranetu.
